## INSTALLATION FICHIERS

### 1. Racine projet

Aucun fichier à copier à la racine.

### 2. Dossier prog/

```batch
cd C:\SiteGITHUB\Hebreu4.0\prog

copy genere_site_v24.0.py genere_site.py
copy cree_table_des_matieres_v6.29.py cree_table_des_matieres.py
copy docx_to_pdf.py docx_to_pdf.py
copy conversion_pdf.py conversion_pdf.py
copy regenerer_tous_pdf.py regenerer_tous_pdf.py
copy corriger_structures_v2.0.py corriger_structures.py
copy generer_page_musique.py generer_page_musique.py
copy lancer_v2.0.cmd lancer.cmd
```

### 3. Dossier prog/lib1/

```batch
cd C:\SiteGITHUB\Hebreu4.0\prog\lib1

copy config_v3.0.py config.py
copy options_v1.1.py options.py
copy html_utils.py html_utils.py
copy structure_utils_v2.0.py structure_utils.py
copy pdf_utils_v1.1.py pdf_utils.py
copy fichier_utils.py fichier_utils.py

```

### 4. Dossier documents/musique/

```batch
cd C:\SiteGITHUB\Hebreu4.0\documents

REM Créer dossier si inexistant
mkdir musique

cd musique
copy entete.html entete.html
copy STRUCTURE.py STRUCTURE.py
```

## FICHIERS À ADAPTER

### prog/lib1/options.py

**Lignes à modifier** :

```python
# LOCAL
BASE_PATH = ""

# GITHUB PAGES
BASE_PATH = "/Hebreu4.0/html"

# Chemins (adapter à votre installation)
DOSSIER_DOCUMENTS = r"C:\SiteGITHUB\Hebreu4.0\documents"
DOSSIER_HTML = r"C:\SiteGITHUB\Hebreu4.0\html"
```

### prog/lib1/config.py

**Ligne à adapter** :

```python
# Pour GitHub Pages
"player_url": "/Hebreu4.0/html/musique/index.html",

# Pour local
"player_url": "/musique/index.html",
```
