1. Contexte et objectif

Le projet Hébreu4.0 est un site statique personnel dédié à l'étude de l'hébreu biblique, hébergé sur GitHub Pages. Son auteur maintient le site à partir de simples fichiers Word (.docx), sans connaissance HTML ni hébergement payant, avec une mise à jour en quelques clics depuis un PC Windows.

4. Contraintes de conception

  • Aucune modification manuelle du HTML généré — tout part des .docx sources
  • config.yaml est le seul fichier à éditer lors d'un changement de poste
  • package\ est la source de vérité — prog\ est toujours régénérable
  • Aucun fichier sensible (token) ne doit être commité sur git
  • Le site doit fonctionner identiquement en local et sur GitHub Pages

6. Hors périmètre

  • Pas d'interface web d'administration
  • Pas de moteur de recherche
  • Pas de génération sur Linux/macOS (PDFCreator = Windows uniquement)
  • Pas de déploiement automatique sans confirmation
